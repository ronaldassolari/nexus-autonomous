# Curso FastAPI Avançado - Resumo

## Módulo 1: Arquitetura
FastAPI é um framework web moderno para Python que usa tipagem e async/await.
Ideal para construir APIs REST de alta performance.

## Módulo 2: Autenticação
JWT (JSON Web Tokens) é o padrão para autenticação stateless.
Usar bcrypt para hash de senhas, nunca armazenar em texto plano.

## Módulo 3: Deploy
Docker é essencial para deploy consistente.
GitHub Actions para CI/CD automático.
Health checks em /health para monitoramento.
